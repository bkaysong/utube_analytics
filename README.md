# 유투브 데이터를 활용한 데이터 분석
----------------------------------
- 팀명 : 3팀
- 팀원 : 김선민, 양정헌, 박영후, 송봉기
- 기간 : 2022.3.14 ~ 2022.4.1

## 프로젝트의 팀원별 역할
>- 김선민 : Elastic 클러스터링구성, Kibana시각화, 웹, ES->HDFS 코드, 로그스태시 설정
>- 양정헌 : Hadoop 구성,
>- 박영후 : 데이터 크롤링
>- 송봉기 : RabbitMQ 구성

## 데이터 분석의 목적
>- 라이브 뉴스 채팅 분석을 통한 우리나라 최근 시사점 파악
